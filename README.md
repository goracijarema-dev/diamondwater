# Діамантова вода

Статичний сайт для GitHub Pages.

## Запуск локально

```bash
python3 -m http.server 8080
```

Після цього відкрити:

```text
http://127.0.0.1:8080
```

## Публікація на GitHub Pages

1. Завантажити всі файли з цієї папки в репозиторій.
2. У GitHub відкрити `Settings` -> `Pages`.
3. Обрати джерело `Deploy from a branch`.
4. Вибрати гілку `main` і папку `/root`.

## Що потрібно додати після запуску бізнесу

- юридичну назву ФОП/ТОВ;
- код ЄДРПОУ або РНОКПП;
- email для звернень;
- банківські реквізити;
- точні умови застави/обміну тари;
- PDF або фото лабораторних аналізів;
- чинні сертифікати, якщо вони є.

## Юридичні сторінки

У сайті вже є внутрішні екрани:

- `#privacy` — політика конфіденційності;
- `#terms` — правила користування;
- `#contract` — публічний договір поставки;
- `#licenses` — ліцензії, сертифікати та аналізи.

Перед публікацією їх потрібно дозаповнити реальними реквізитами продавця та документами. Орієнтири:

- Закон України "Про захист персональних даних": https://zakon.rada.gov.ua/laws/show/2297-17
- Закон України "Про електронну комерцію": https://zakon.rada.gov.ua/laws/show/675-19
- Закон України "Про захист прав споживачів": https://zakon.rada.gov.ua/laws/show/1023-12
